# CareLens AI

CareLens AI is an AI-powered healthcare assistant built during a 30-hour hackathon using the MERN stack. The platform helps users monitor their health, analyze reports, and receive AI-driven health guidance through various smart modules.

## Overview

CareLens AI provides a centralized dashboard where users can access multiple AI healthcare tools including symptom analysis, diet planning, sleep advice, skincare analysis, and report analysis. The platform also includes personal health modules and family health management.

## Tech Stack

Frontend:

* React.js
* Tailwind CSS
* Framer Motion

Backend:

* Node.js
* Express.js

Database:

* MongoDB

AI Integration:

* Groq API

## Installation

Clone the repository

```
git clone https://github.com/jeslipriya/AI-HEALTHCARE
```

Go to the project directory

```
cd AI-hEALTHCARE
```

Install dependencies

```
npm install
```

Start the development server

```
npm run dev
```
